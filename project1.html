Image Editting Application
Sumarry
A small project to build a simple image editing application, created to test and improve my Python skills. This project includes a graphical user interface (GUI) built using Tkinter.
Objective
The main goal of this project is to create a user-friendly application that allows users to perform basic image editing tasks such as cropping, resizing, and applying filters. This project aims to enhance my programming skills and gain practical experience in GUI development.
Skills and Tools
- Programming Language:Python
- GUI Library:Tkinter
- Image Processing Library: PIL (Pillow)
- Version Control: Git and GitHub
- Development Environment: Visual Studio Code
